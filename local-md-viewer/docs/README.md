# Welcome to Folio

This is your local markdown document viewer. Folio provides a beautiful, editorial-style reading experience for your markdown files.

## Features

- **Clean Typography**: Beautiful serif headings with readable sans-serif body text
- **Dark Mode**: Easy on the eyes with a warm, paper-like reading area
- **File Browser**: Navigate your documents with an intuitive sidebar
- **Search**: Quickly find documents by name
- **Live Updates**: Changes to your markdown files are reflected automatically

## Getting Started

1. Add your markdown files to the `docs` folder
2. Or set `DOCS_DIR` to point to your existing documentation
3. Run `bun run dev` to start the viewer

## Supported Markdown Features

Folio supports all standard markdown features:

- **Bold** and *italic* text
- [Links](https://example.com)
- Lists (ordered and unordered)
- Code blocks with syntax highlighting
- Tables
- Blockquotes
- And more!
